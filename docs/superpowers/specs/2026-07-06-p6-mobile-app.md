# KEZA P6: Mobile App (React Native / Flutter)

**Goal:** Native iOS/Android apps with same functionality as web.

**Scope:** React Native (preferred) or Flutter wrapper around existing APIs.

**Features:**
- Flight search with all filters
- Miles program selection
- Price comparison (cash vs miles)
- Price alerts (push notifications)
- Offline mode (cached data)
- Deep linking from web (share searches)

**Architecture:**
- Leverage existing Next.js APIs
- React Native frontend
- Firebase for push notifications
- Realm DB for offline storage

**Effort:** Very High (3-4 sprints)

**Status:** Spec only, defer to next quarter.

**Success Criteria:**
- ✅ iOS app on App Store
- ✅ Android app on Google Play
- ✅ Feature parity with web
- ✅ 10k+ downloads in first month
