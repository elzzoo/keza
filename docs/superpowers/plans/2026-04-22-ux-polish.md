**Changement 3 — Table overflow sur mobile** (ligne ~181 environ, autour du `<div className="bg-surface border border-border rounded-2xl overflow-hidden">`).

Trouver ce bloc :
```tsx

  <table className="w-full text-sm">
```

Le remplacer par :
```tsx

  <div className="overflow-x-auto">
    <table className="w-full text-sm min-w-[400px]">
```

Et fermer le `<div className="overflow-x-auto">` juste avant le `</div>` du container external :
```tsx

  </div>
</div>
```
